# POL
A polarization defect detection dataset for industrial defect inspection

POL Dataset
If you find our work useful for your researc, please ggive us a STAR!

Preview

Our dataset contains 424 polarization images, of which the training and test sets contain 339 and 85 images respectively. It covers six industrial scenarios including metal, blots, plastics, ironwork, paper, and glass. Each sample provides multi-modal polarization information (intensity, angle of polarization (AoP), and degree of polarization (DoP)), enabling comprehensive evaluation of defect detection models under various material conditions.
